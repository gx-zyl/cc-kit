| Skill | 来源 | 用途 |
|-------|------|------|
| diagnose | mattpocock/skills | 6 阶段调试法 |
| grill-with-docs | mattpocock/skills | 用文档拷打计划 |
| tdd | mattpocock/skills | TDD 红-绿-重构 |
| caveman | mattpocock/skills | 极简输出省 Token |
| handoff | mattpocock/skills | 交接文档 |
| grill-me | mattpocock/skills | AI 拷打设计 |
| write-a-skill | mattpocock/skills | 创建自定义 Skill |
| karpathy-guidelines | multica-ai/andrej-karpathy-skills | LLM 编码行为指南 |
| huangting-protocol | XianDAO-Labs/huangting-protocol | 任务执行生命周期管理 |

## 安装

放在 `~/.claude/plugins/cc-kit/` 即可被 Claude Code 自动发现。
